# React Hooks Application de liste Todo
> une application List TODO simple utilisant ** React Hooks,  ** et ** Material-UI **

## Prérequis

- Node 12.18.0 ;

- npm 7.5.2 ;

## Installer
Clonez le repo   dans votre local. Entrez dans le dossier de  projet et exécutez
`` sh
npm ci/ npm install

## Run 

```sh
npm start
```
